How a Web Page is Assembled

A web page consists of two main parts: a head and a body. The body is the content, it is what shows up on the screen. The head, however, is where the configuration for the page is stored. The browser uses the head section to get its instructions for HOW that content should be displayed. This can include the CSS (Cascading Style Sheets) which tell the browser about the appearance of the text. It can also include scripts that will run code on a page. You can always recognize these in a webpage's code because they look like this: &lgt;script src="filename.js"></script>. This points to another file which will contain code that runs on the webpage to give it its extra functionality.

Logic

A computer runs by making a series of logical decisions, much like a human does. A human might say, "If it is raining, use your umbrella." Where a computer will say, "If the value stored in x is equal to 10, output 'Ten'"

Study the code that is included in the webpage, and see if you can figure out where an if statement is testing some logical condition which would make the bomb explode.

Tip: You can stop any line of code from running by putting // in front of the line. That comments it out and tells the computer to ignore it.
